# Contributing

Use Java 21 and target Paper 1.21.11. Keep user-facing text in locale YAML files
and render it through MiniMessage. New features must include focused tests and an
update to `docs/PARITY.md`.

Publicly observable behavior may be used as a clean-room specification. Do not
copy proprietary code, maps, textures, branding, forum posts, exam questions, or
configuration from another server.
